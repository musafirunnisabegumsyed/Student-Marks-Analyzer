# 📊 Student Marks Analyzer

## 📌 Overview
The Student Marks Analyzer is a Python menu-driven project that manages student records and performs analysis such as average, highest/lowest marks, percentage calculation, and topper identification.

---

## 🚀 Features
- Add and store student names with marks  
- View all student records  
- Search student by name  
- Calculate individual student average  
- Find highest and lowest marks  
- Calculate percentage of students  
- Generate leaderboard (Topper system)  
- Input validation and error handling  

---

## 🛠️ Technologies Used
- Python  
- Lists & Tuples  
- Functions  
- Loops  
- Conditional Statements  
- Exception Handling  
- Sorting logic (manual implementation)
